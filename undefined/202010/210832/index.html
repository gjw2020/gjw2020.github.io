MarkdownPad2基础语法

苦于言语，穷于学识。该篇主要用于MarkdownPad2语法学习，和博客的书写。格式方面有待提高！
相关内容借鉴简书Markdown语法。如有冒犯，还请告知。

标题设置

语法：

# 一级标题

## 二级标题

### 三级标题

#### 四级标题

##### 五级标题

####### 六级标题

效果：

一级标题

二级标题

三级标题

四级标题

五级标题
六级标题

字体设置

  • 加粗

语法：

**加粗内容**

效果：

加粗内容

  • 斜体

语法：

*斜体文字*

效果：

斜体内容

  • 斜体加粗

语法：

***斜体加粗***

效果：

斜体加粗

引用

语法：

> 第一层应用

>> 第二层引用

>>>第三层应用

>>>>第四层引用
...

效果：

第一层应用

第二层引用

第三层应用

第四层引用

分割线

语法：

---

效果：

图片

语法：

1|![图片alt](图片地址 "图片title")    
2|# 图片地址可以为网页地址也可以是本地地址。
3|# 图片alt是显示在图片下面的文字。
4|# 图片title是图片标题，当鼠标移到图片上时显示的内容。

示例：

# 输入网页地址

超链接

语法：

1|[超链接名](超链接地址 "超链接title")
2|title可加可不加

示例：

[我的博客](https://flyingdiary.top)

效果：

我的博客

列表

无序列表

语法：

+ 列表内容1
+ 列表内容2
+ 列表内容3

效果：

  • 列表内容1
  • 列表内容2
  • 列表内容3

有序列表

语法：

1|1. 列表内容1
2|2. 列表内容2
3|3. 列表内容3
4|
5|注意：序号根内容之间要有空格

效果：

  1. 列表内容1
  2. 列表内容2
  3. 列表内容3

列表嵌套

上一级和下一级之间敲三个空格

示例：

+ 一级无序列表内容    
   + 二级无序列表内容
   + 二级无序号列表内容
      + 三级无序列表内容
      + 三级无序列表内容

效果：

  • 一级无序列表内容
    • 二级无序列表内容
    • 二级无序号列表内容
      • 三级无序列表内容
      • 三级无序列表内容

表格

语法：

表头|表头|表头
.---|:--:|---:
内容|内容|内容
内容|内容|内容

#|充当分割表格作用
#第二行分割表头和内容。
#- 有一个就行，为了对齐，多加了几个
#文字默认居左
#-两边加：表示文字居中
#-右边加：表示文字居右

效果：

表头 表头 表头
内容 内容 内容
内容 内容 内容

区块

  • 单行代码

语法：

`print("Hello,World!")`

#用两个反斜杠

效果：

print("Hello,World!")

  • 代码段

语法：

1|    def function(r):
2|        squre = 2*pi*r
3|            return squr
4|        
5| # 代码整体缩进

效果：

def function(r):
   squre = 2*pi*r
   return squr

总结：

1.注意语句之间大多需要换行

2.单个语法符号与内容之间大多需要空格
